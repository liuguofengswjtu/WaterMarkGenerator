# 照片水印生成器

根据照片 EXIF 信息自动生成文字水印的桌面应用程序。

## 功能特点

- **自动读取 EXIF**：自动提取相机型号、镜头型号、光圈、快门速度、ISO、焦距等拍摄参数
- **优雅水印样式**：参考专业摄影水印风格，三行文字层次分明
- **丰富的自定义选项**：
  - 自由调节字体类型
  - 分别设置三行文字的字号
  - 调节文字透明度
  - 斜体效果
  - 行间距与边距控制
- **灵活的位置控制**：
  - 滑条精确调节水平/垂直位置
  - 支持在预览图上直接拖动水印
- **批量处理**：支持单文件、多文件、整个文件夹批量添加水印
- **保留原图信息**：输出图片保留原始 EXIF 数据，不改变其他内容

## 运行环境

- **Windows**: Windows 7/10/11
- **macOS**: macOS 10.14+ (Intel / Apple Silicon)
- Python 3.8+
- 依赖库：PyQt5, Pillow, piexif

## 安装与运行

### 方式一：直接运行源码

1. 确保已安装 Python 3.8 或更高版本
2. 安装依赖：
   ```bash
   pip install PyQt5 Pillow piexif
   ```
3. 运行程序：
   ```bash
   python main.py
   ```

> **macOS 注意**：首次运行若提示无法打开，请在"系统偏好设置 → 安全性与隐私"中允许该应用运行。

### 方式二：打包为可执行文件（可选）

#### Windows

```bash
pip install pyinstaller
python build_exe.py
```

打包后的 `.exe` 文件位于 `dist/照片水印生成器.exe`。

#### macOS

```bash
pip install pyinstaller
python build_exe.py
```

打包后的 `.app` 应用包位于 `dist/照片水印生成器.app`，可直接拖入"应用程序"文件夹使用。

> **字体差异**：macOS 系统默认提供"苹方"、"黑体-简"等中文字体，界面会自动适配可用字体。

## 使用说明

### 基本步骤

1. **选择图片**
   - 点击 **"选择图片"** 选择单个或多个文件
   - 或点击 **"选择文件夹"** 导入整个文件夹内的图片

2. **调整水印样式**
   - 在右侧面板选择字体、调节字号、透明度等
   - 第一行通常为相机型号大字，第二、三行为技术参数

3. **预览与定位**
   - 左侧实时预览水印效果
   - 可用 **滑条** 调节位置，也可以 **直接在预览图上拖动** 水印

4. **开始处理**
   - 点击 **"开始处理"**，选择输出文件夹
   - 程序将批量处理所有选中的图片

### 参数说明

| 参数 | 说明 |
|------|------|
| 字体 | 系统中已安装的字体，默认微软雅黑 |
| 第一行字号 | 相机型号展示字号，建议 36-60 |
| 第二行字号 | 技术型号字号，建议 16-24 |
| 第三行字号 | 拍摄参数字号，建议 16-24 |
| 透明度 | 0（全透明）~ 255（不透明），建议 180-220 |
| 水平/垂直位置 | 水印左上角相对图片的位置百分比 |
| 行间距 | 三行文字之间的间距（像素） |
| 边距 | 水印文字到定位点的内边距 |

### 水印格式

根据 EXIF 信息自动生成：

```
第一行: Sony α1 II        (品牌化相机名称)
第二行: ILCE-1M2 / SEL600F40GM   (技术型号 / 镜头型号)
第三行: ISO400 / 1/2500s / F4 / 600mm  (拍摄参数)
```

### 注意事项

- 如果照片没有 EXIF 信息，水印将显示 `Unknown Camera`
- 输出文件会在原文件名后添加 `_watermarked` 后缀
- 支持的格式：JPG、JPEG、PNG、BMP、TIFF、WebP
- 处理后的图片会尽量保持原始格式和质量
