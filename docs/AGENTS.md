# 实现一个照片水印生成器，可以给照片添加文字水印  
功能要求：  
1. 核心功能：是根据照片的EXIF信息，将照相机型号，镜头型号，拍摄信息（光圈、快门、ISO），以参考图"参考文字水印.png"的格式添加到照片上，除此之外，不改变原有照片的任何其他信息，包括格式和内容都维持原样。
2. 用户界面要求：在Windows环境下运行，具有用户友好以及简洁优雅的GUI，能够让无编程背景的用户也轻松使用。  
    1. GUI可以分为左侧预览区和右侧功能操作区，预览区显示所选图片，图片外背景为淡白色。  
    2. GUI需要包含能够自由、分行调节水印每一行文字字体类型、字体大小、字体间距、字体透明度、是否斜体等的功能。  
    3. GUI需要能够实现自由调节水印位置的功能，以滑条拖动或者直接将水印拖动的方式实现。  
    4. GUI需要实现对索尼阿尔法LOGO的特殊处理，将第一行首个阿尔法字母替换为我给你的“new_sony_logo_centeraligned.svg”，并与该行文字呈上下居中对齐。  
    5. GUI需要实现对阿尔法LOGO的大小，透明度的调节。  
    6. GUI需要能够加入用户自定义水印，也可自由调整大小，位置，和透明度，通过滑条和直接拖拽两种方式实现。  
3. 输入文件选择功能：需要能够让用户选择需要添加水印的文件的功能；需要考虑到，可以是单个文件，也可以是多个文件，也可以是文件夹。   
4. 输出文件路径选择功能：需要让用户能够选择输出添加好水印后的文件的路径。  
5. 需要对程序可能出现的异常的提升和处理，对于不存在EXIF信息的图片，不进行处理。  
6. 需要添加处理中的进度条，并且给出实时进度，在未处理完时，不允许新添加水印。  
7. 在处理完成之后，显示一个处理结果窗口，窗口需要显示成功和失败的文件数目，并显示处理失败的文件的名称和原因。  
8. 可以使用任何合适的语言实现。  
9. 需要有详细的软件使用说明文档。
